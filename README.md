## Como rodar o projeto

1 - Clone o repositório
```sh
git clone https://github.com/DarkMetaK/clone-dio.git
```

2 - Instale as dependências
```sh
npm install
```

3 - Execute o projeto
```sh
npm run dev
```

4 - Execute o Storybook
```sh
npm run storybook
```
